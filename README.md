## 📖 Introduction

This project is a modern reimagining of the classic **Flappy Bird** game, but with a twist: **No keyboard or mouse required!** Instead of tapping the screen, players control the bird using **Hand Gestures** captured via the webcam. This project demonstrates the application of **Computer Vision** and **Human-Computer Interaction (HCI)** concepts using Python.

## 🚀 Key Features

* **Real-time Hand Tracking:** Utilizes Google's MediaPipe framework to detect hand landmarks with high accuracy and low latency.
* **Gesture Control:** The bird jumps based on specific hand movements (e.g., pinching fingers / raising hand).
* **Smooth Gameplay:** Optimized frame rate for a seamless experience on standard laptops.
* **Classic Mechanics:** Familiar scoring system, collision detection, and increasing difficulty.

## 🛠️ Tech Stack

* **Language:** Python
* **Game Engine:** Pygame
* **Computer Vision:** OpenCV (`cv2`)
* **AI/ML Model:** MediaPipe (Hand Solutions)
